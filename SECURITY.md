# Security Policy

## Supported Versions

We only support and patch the latest major release version of `@regexto/validators`.

| Version | Supported          |
| ------- | ------------------ |
| < 1.x   | :x: No             |
| >= 1.x  | :white_check_mark: Yes |

## Reporting a Vulnerability

If you discover a security vulnerability within this package, please do **NOT** open a public issue. Instead, report it privately to ensure the safety of our users.

Please report security issues by emailing: **security@regex.to** (or directly contacting Łukasz Oleniuk at **lukasz@oleniuk.pl**).

We will acknowledge your report within 48 hours and work with you to coordinate a security patch and release before public disclosure.
